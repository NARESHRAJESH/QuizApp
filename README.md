# QuizApp 🎯

A React-based Quiz Application that allows users to take interactive quizzes with real-time scoring, timer, and progress tracking.

---

## 🚀 Live Demo
https://naresh-react-quiz-app.netlify.app

---

## 📌 Features

- Add / Edit / Delete Quiz Questions (CRUD)
- Start Quiz Mode
- Timer for each question (15 seconds)
- Score calculation
- Progress bar
- High score saved using Local Storage
- Responsive UI

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- Vite
